# 算法题-刷题技巧

> ### 切题四件套
>
> - Clarification：读题，确认清楚题目的意思；
> - Possible solutions：要想多个解法，不要用第一个思路来解题；
>   - compare（time/space）：降低时间复杂度
>   - optimal：优化算法，找到最优解；
> - Coding：多写
> - Test cases：测试用例

### 刷题第一遍

1. 5分钟：读题 + 思考；
2. 直接看解法：注意！多解法，比较解法优劣；
3. 背诵、默写好的解法。

### 

### 刷题第二遍

1. 马上自己写 ---->  LeetCode 提交，直到通过；
2. 多种解法比较、体会 ----> 优化。（主要看时间）



### 刷题第三遍

1. 过了一天后，再重复做题；
2. 不同解法的熟练程度 ----> 专项练习



### 刷题第四遍

- 过了一周：反复回来练习相同题目。



### 刷题第五遍

- 面试前一周或者半个月再做一次。
