# [206-面试题24. 反转链表](https://leetcode-cn.com/problems/fan-zhuan-lian-biao-lcof/)

```js
var reverseList = function(head) {
    let pointer = null, curr = head, temp;
  	while (curr) {
      temp = curr.next;

      curr.next = pointer;
      pointer = curr;

      curr = temp
    }
  	return pointer;
};
```

这个题的理解，在链表继续的过程中，做处理。

链表的遍历：

```js
while (list) {
  list = list.next;
}
```

想要在链表的遍历做处理，就需要把这个过程拆开

```js
while (list) {
  temp = list.next;
  // handle listNode
  list = temp;
}
```

使用ES6的解构赋值

```js
var reverseList = function(head) {
    let [prev, current] = [null, head]
    while(current) {
        [current.next, prev, current] = [prev, current, current.next]
    }
    return prev
}
```

转义ES5

```js
"use strict";

var reverseList = function reverseList(head) {
  var prev = null,
      current = head;

  while (current) {
    var _ref = [prev, current, current.next];
    current.next = _ref[0];
    prev = _ref[1];
    current = _ref[2];
  }

  return prev;
};
```
